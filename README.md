# Create and Publish Skills

A skill that guides you through creating and publishing agent skills to GitHub.

## Installation

```bash
npx skills add ffhope/create-publish-skill@create-publish-skill
```

## What This Skill Does

This skill provides step-by-step instructions for:
- Creating standard skill files (SKILL.md)
- Setting up GitHub repositories
- Publishing skills for installation via `npx skills add`
- Understanding skill requirements and best practices

## Quick Start

1. Read SKILL.md for complete instructions
2. Create your skill following the template
3. Push to GitHub (must be Public)
4. Install with `npx skills add`

## Requirements

- GitHub account
- Public repository
- SKILL.md file with YAML frontmatter
- Proper naming and description format

## Resources

- Skills Directory: https://skills.sh/
- Skills CLI: `npx skills --help`
- Find skills: `npx skills find <keywords>`
